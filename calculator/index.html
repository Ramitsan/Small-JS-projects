Инструкция
  • Для извлечения квадратного корня наберите число, например, 25, затем нажмите кнопку со знаком √.
  • Для ввода отрицательного числа наберите число, например, 5, затем нажмите кнопку со знаком +\-.
  • Для извлечения квадратного корня из отрицательного числа наберите число, например, 9, затем нажмите кнопку +\-, а затем - кнопку √.
  • Для операций с отрицательными числами вначале наберите число, например, 10, затем нажмите кнопку +\-, затем - кнопку со знаком операции,например, *, и далее второе число, например, 2.
  • Для возведения в степень наберите число, например, 3, затем нажмите кнопку со знаком ^, затем - показатель степени, например, 2.
